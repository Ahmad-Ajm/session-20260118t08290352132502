# ملخص المحادثة - session-20260118t08290352132502

1) Vision
- حل مشكلة صعوبة كتابة السيرة الذاتية (CV) وإيصالها للمهتمين عبر تمكين إنشاء CV بسهولة من خلال جمع المعلومات وتنظيمها في ملف، مع تخزينها في قاعدة بيانات للعرض والبحث، مع الحفاظ على الخصوصية عند المشاركة/التصدير.

2) Personas
- الأفراد (مستخدمون مسجّلون): ينشئون/يعدّلون سيرهم الذاتية.
- الزوار (غير مسجّلين): يستعرضون السير الذاتية المتاحة ويبحثون/يصفّون ويصدرون نسخة مع إخفاء البيانات الحساسة.

3) Features

A) In-scope (Now)
- FEAT-001-"UX/UI (Baseline/System)"
- FEAT-002-"نموذج إدخال CV بصفحة واحدة"
- FEAT-003-"حقول: الاسم"
- FEAT-004-"حقول: الموبايل"
- FEAT-005-"حقول: الصورة"
- FEAT-006-"حقول: الخبرات"
- FEAT-007-"حقول: التعليم"
- FEAT-008-"تسجيل المستخدمين لإنشاء/تعديل CV"
- FEAT-009-"إنشاء CV من البيانات المدخلة"
- FEAT-010-"تخزين السير الذاتية بقاعدة بيانات للعرض والبحث"
- FEAT-011-"عرض جميع السير الذاتية المتاحة"
- FEAT-012-"تصدير CV للزوار مع إخفاء البيانات الحساسة"
- FEAT-013-"بحث سريع في السير الذاتية"
- FEAT-014-"تصفية سريعة للسير الذاتية"

B) Out-of-scope / Deferred (User said later)
- لا يوجد مذكور.

4) Data / Content
- بيانات السيرة الذاتية المدخلة: الاسم، رقم الموبايل، صورة شخصية، الخبرات، التعليم.
- بيانات حساب المستخدم للتسجيل (لم تُحدد الحقول/الطريقة).
- بيانات معروضة للزوار ضمن قائمة السير الذاتية وداخل CV.
- بيانات “حساسة” يتم إخفاؤها عند التصدير (لم يتم تحديد ما هي الحقول الحساسة تحديدًا).

5) Non-Functional Requirements
- سرعة التحميل والتصفح.

6) Open / Missing Information
- منصة التطبيق: ويب/جوال/كلاهما.
- طريقة تسجيل المستخدمين وحقول الحساب (بريد/هاتف/كلمة مرور…).
- هل يوجد أدوار إضافية مثل Admin لإدارة المحتوى/المستخدمين.
- ما المقصود بـ “عرض جميع السير الذاتية المتاحة”: هل كل CV عامة افتراضيًا أم هناك إعداد خصوصية/نشر.
- تعريف “البيانات الحساسة” التي يجب إخفاؤها عند التصدير (مثلاً: الموبايل؟).
- صيغة التصدير المطلوبة (PDF/طباعة/رابط مشاركة…).
- تفاصيل البحث والتصفية (حسب الاسم/التعليم/الخبرات…).
- آلية رفع/تخزين الصورة (حجم/نوع/مكان التخزين).
- هل يسمح للمستخدم بامتلاك أكثر من CV أم CV واحد لكل حساب.
- اللغات المطلوبة (عربي فقط أم متعدد اللغات).
- متطلبات أمن/خصوصية إضافية غير مذكورة (صلاحيات الوصول، تشفير، إلخ).
